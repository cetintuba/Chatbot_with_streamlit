# Streamlit Chatbot

This project is a simple chatbot application built with **Streamlit** and developed using **Google Colab**.

## Description
The chatbot provides a basic chat interface where users can interact with the system.  
This project is created for learning purposes and to practice chatbot development.

## Features
- Simple chatbot interface
- Built with Streamlit
- Developed and tested on Google Colab
- Beginner-friendly project

## Technologies Used
- Python
- Streamlit
- Google Colab

## How to Run
1. Open the project in Google Colab  
2. Install the required libraries  
3. Run the Streamlit application  

## Purpose
The main purpose of this project is to learn Streamlit and understand the basics of chatbot systems.
